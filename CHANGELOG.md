# Changelog

## v0.1.0 (2026-03-19)

Initial release — Quick Translate + Session-based translation.

### Features

- Instant Translation: Quick translate bar, Auto Detect, 59 languages
- Translation Sessions: Chat-style, create/delete/edit/export(MD)
- Summarization: Separate model, SSE streaming progress
- Token Streaming: Real-time progress bar (Ollama stream → SSE)
- Chunking: Configurable XSmall(100) ~ XLarge(5,000)
- Smart Model Management: Auto unload/load to prevent memory overflow
- Language Detection: langdetect auto-detect
- LLM Settings: Multi-provider (Ollama/DeepL/Google/OpenAI), separate translate/summarize models
- UI Language: KO/EN/ZH with cookie persistence
- Dark Mode, Copy/Summary/Clear, Undo/Clear All
