# Localingo 개발상세서 (간략)

> 작성일: 2026-03-19

---

## 1. 개요

**한 줄 요약**: Ollama 기반 로컬 번역 + 요약 도구 (채팅형)

**문제 → 해결**: 번역할 때마다 Google Translate나 DeepL에 데이터를 보내야 함 → 로컬 Ollama로 프라이버시 유지하며 번역+요약

**기술 스택**: FastAPI + Next.js + SQLite + Tailwind (경량, 인증 없음)

---

## 2. 사용자 및 역할

인증 없음. 단일 사용자. 개인 로컬 도구.

## 3. 기능

| # | 기능 | 설명 | 우선순위 |
|---|------|------|----------|
| F01 | 채팅형 번역 | 텍스트 입력 → 번역 결과 말풍선 누적 | P0 |
| F02 | 언어 감지+선택 | 자동 감지 → 타겟 선택 (기본 한국어) + 스왑 | P0 |
| F03 | 단일 파일 번역 | TXT/MD/DOCX/PDF 업로드 → 번역 → 다운로드 | P0 |
| F04 | 번역 이력 | SQLite 저장, 목록 조회, 검색 | P0 |
| F05 | 이력 선택 요약 | 체크박스로 이력 선택 → qwen3.5로 요약 | P1 |
| F06 | 다중 파일 배치 번역 | 여러 파일 동시 업로드 → 진행률 표시 | P1 |
| F07 | 배치 결과 통합 요약 | 배치 번역 결과 전체 → 요약 | P2 |
| F08 | 파일 포맷 변환 | 번역 결과를 다른 포맷으로 다운로드 | P2 |
| F09 | Ollama 상태 표시 | 연결 여부 + 모델 목록 | P0 |
| F10 | 모델 오버라이드 | 설정에서 드롭다운으로 모델 변경 | P1 |

## 4. 데이터 (SQLite)

```sql
CREATE TABLE translations (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  source_text TEXT NOT NULL,
  translated_text TEXT NOT NULL,
  source_lang TEXT NOT NULL,
  target_lang TEXT NOT NULL,
  model TEXT NOT NULL,
  task_type TEXT DEFAULT 'translate',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE file_translations (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  original_filename TEXT NOT NULL,
  original_path TEXT,
  translated_path TEXT,
  source_lang TEXT NOT NULL,
  target_lang TEXT NOT NULL,
  model TEXT NOT NULL,
  status TEXT DEFAULT 'pending',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE settings (
  key TEXT PRIMARY KEY,
  value TEXT NOT NULL
);
```

## 5. API

| Method | Path | 설명 |
|--------|------|------|
| GET | /api/health | 상태 확인 |
| GET | /api/ollama/status | Ollama 연결 + 모델 목록 |
| GET | /api/languages | 지원 언어 목록 |
| POST | /api/translate | 텍스트 번역 |
| POST | /api/summarize | 텍스트 요약 |
| POST | /api/translate/file | 단일 파일 번역 |
| POST | /api/translate/batch | 다중 파일 배치 번역 |
| GET | /api/history | 번역 이력 (페이지네이션) |
| DELETE | /api/history/{id} | 이력 삭제 |
| POST | /api/history/summarize | 선택 이력 요약 |
| GET | /api/settings | 설정 조회 |
| PUT | /api/settings | 설정 변경 |

인증: 전부 X (로컬 전용)

## 7. 구현 순서

```
Phase 0: 경량 템플릿 생성 (FastAPI + Next.js, 인증 없음)
Phase 1: 백엔드 — Ollama 연동, 번역 API, SQLite
Phase 2: 메인 UI — 채팅형 번역 + 언어 선택
Phase 3: 파일 번역 — 업로드/다운로드/포맷변환
Phase 4: 이력 + 요약 — 이력 목록, 선택 요약
Phase 5: 배치 번역 + 설정 + 마무리
```

## 모델 자동 선택

| 작업 | 모델 | 이유 |
|------|------|------|
| 번역 | translategemma:latest | 번역 특화 |
| 요약 | qwen3.5:latest | 범용 LLM |
| 언어 감지 | Python langdetect | 경량 |

설정에서 고급 사용자가 드롭다운으로 모델 오버라이드 가능.
