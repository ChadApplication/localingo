# Localingo — 아이디어노트



## 핵심 아이디어

Ollama `translategemma:latest` (3.3GB) 기반의 **로컬 다기능 번역 + 문서 처리 앱**. 인터넷 없이, 비용 없이, 프라이버시를 유지하며 다국어 문서 작업.

---

## 기능 (구상)

### 번역
- 텍스트 번역 (KO/EN/ZH/JA + 기타)
- 파일 번역 (PDF, DOCX, TXT, MD, CSV)
- 클립보드 번역 (복사 → 자동 번역)
- 배치 번역 (폴더 내 다수 파일)

### 문서 처리
- 요약 (긴 문서 → 핵심 요약)
- 포맷 변환 (MD ↔ DOCX ↔ PDF)
- 용어 통일 (glossary 기반 일관된 번역)
- 비교 보기 (원문 / 번역문 나란히)

### 연구자 특화 (확장)
- 논문 초록 번역 + 정리
- 참고문헌 다국어 관리
- 연구 노트 다국어 작성

---

## 기술 스택

| 구성 | 기술 |
|------|------|
| LLM | Ollama translategemma:latest (3.3GB) / translategemma:27b (17GB) |
| 백엔드 | FastAPI (Python) |
| 프론트엔드 | Next.js (chad-app-template 기반) |
| 문서 처리 | python-docx, PyPDF2, pandoc |
| 로컬 실행 | Docker 없이 직접 실행 |

## 차별점

- **완전 로컬**: 데이터가 외부로 나가지 않음
- **무료**: Ollama 기반, API 비용 없음
- **다기능**: 번역만이 아닌 문서 처리 통합
- **연구자 친화**: 학술 문서 특화 기능
