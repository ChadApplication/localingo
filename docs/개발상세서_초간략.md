# Localingo 개발상세서 (초간략)

---

## 앱 이름

Localingo

## 한 줄 설명

Ollama 기반 로컬 번역 + 요약 도구 (채팅형 인터페이스)

## 누가 쓰나?

나 혼자 (인증 없음, 개인 로컬 도구)

## 무엇을 할 수 있나?

- [x] 언어 감지 → 타겟 언어 선택 (기본: 한국어) + 방향 스왑
- [x] 채팅형 번역 (텍스트 입력 → 번역 결과가 말풍선처럼 쌓임)
- [x] 번역 이력에서 선택 → 요약
- [x] 단일 파일 업로드 → 번역 → 포맷 변환 다운로드
- [x] 여러 파일 동시 번역 (배치)
- [x] 여러 파일 번역 결과 → 통합 요약
- [ ] ~~회원가입 / 로그인~~ (불필요)

## 화면 구성

| 화면 이름 | 설명 |
|-----------|------|
| 번역 (메인) | 채팅형 인터페이스. 텍스트 입력 → 번역 결과 말풍선. 상단: 언어 선택 + 스왑 |
| 파일 번역 | 파일 드래그앤드롭 (단일/다중). 번역 진행률. 포맷 변환 다운로드 |
| 이력 | 번역 히스토리 목록. 체크박스 선택 → 요약 생성 |
| 설정 | Ollama URL, 모델 오버라이드 (고급), 기본 언어 |

## 데이터

- SQLite (백엔드)
- 번역 이력: 원문, 번역문, 소스언어, 타겟언어, 시간, 모델, 작업유형
- 파일 번역 결과: 원본 파일명, 번역 파일 경로, 상태
- 설정: Ollama URL, 기본 언어쌍

## 모델 (자동 선택)

| 작업 | 모델 | 이유 |
|------|------|------|
| 번역 | translategemma:latest | 번역 특화 |
| 요약 | qwen3.5:latest | 범용 LLM |
| 언어 감지 | Python langdetect 또는 translategemma | 경량 |

설정에서 고급 사용자가 모델 오버라이드 가능.
